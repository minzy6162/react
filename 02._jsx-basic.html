  • render 메소드를 호출하여 실제 DOM에 내용을 출력

  • 왜 React.render가 아니라 ReactDOM.render인가?

    • 리액트 요소가 DOM에만 그려지지는 않기 때문에 (ex: 리액트 네이티브) 따로 렌더링을 하는 모듈을 분리하여 사용하게 되었으므로 ReactDOM.render를 사용하여 DOM 구성 (기존 React.render는 사용 중지됨)

    render() {
        const mystyle = {
